Gloss 隐私政策

最后更新：2025-06-05

Gloss（"我们"）非常重视您的隐私。本政策说明 Gloss Chrome 扩展（"扩展"）收集哪些信息、如何使用这些信息以及您的权利。

1. 数据收集

Gloss 不收集任何个人数据。具体而言：
  • 不收集个人身份信息（姓名、邮箱等）
  • 不收集浏览历史
  • 不收集点击行为或使用习惯
  • 不使用 Cookie 或任何追踪技术
  • 不与任何第三方共享数据

2. 本地存储

扩展使用 Chrome 的 chrome.storage.local API 将以下数据保存在您的浏览器本地：
  • 用户设置：API 地址、模型名称、API Key、主题偏好、语言偏好
  • 释义缓存：已查询过的术语和 AI 返回的释义，避免重复调用
  • 使用统计：查询次数、缓存命中数等，仅用于本地展示
  • 允许复制网站列表：用户手动启用的域名白名单

以上所有数据仅保存在您的浏览器中，不会上传到任何服务器。卸载扩展后，这些数据将一并删除。

3. AI 服务调用

当您选中文本并触发查询时，扩展会将以下内容发送到您自行配置的 AI API 服务：
  • 选中的术语文本
  • 术语周围的上下文段落（用于提供语境释义）

Gloss 不运营任何 AI 服务。您选择的 API 提供商（如 DeepSeek、OpenAI 等）受其各自的隐私政策约束。请查阅您所使用服务的隐私政策以了解其数据处理方式。

4. 权限说明

扩展请求以下浏览器权限，各权限的用途如下：
权限用途
storage保存用户设置和释义缓存
activeTab获取当前页面的选中文本和上下文
scripting在用户启用时注入 CSS 以解除文本选择限制
alarms定时检查版本更新
host: <all_urls>在任意网页上检测文本选择并显示释义提示

5. 版本更新检查

扩展会定期通过 HTTPS 访问 maidou0215.github.io/gloss-pages/version.json 检查是否有新版本。此请求不包含任何用户数据，仅获取最新版本号。

6. 儿童隐私

Gloss 不面向 13 岁以下儿童，也不会有意收集儿童的个人信息。

7. 政策变更

我们可能会不时更新本隐私政策。重大变更将通过扩展更新说明或 GitHub 仓库公告通知。继续使用扩展即表示您同意修订后的政策。

8. 联系方式

如有隐私相关问题，请通过 Chrome Web Store 扩展详情页与我们联系。
